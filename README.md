# Open Source Contributions

**[Hinglish Whisper Benchmark](https://github.com/weekendpm/hinglish-whisper-benchmark)** · [openai/whisper #2761](https://github.com/openai/whisper/discussions/2761)

Evaluated OpenAI's whisper-1 on 20 Indian-context audio clips across 6 categories — pure Hindi, Hinglish, Indian proper nouns, addresses with PIN codes, and call-center phrases.

Key finding: **no output path exists for romanized Hinglish.** Whisper switches the output script entirely — Latin-script input returns as Devanagari. Structural blocker for Indian support-call and WhatsApp transcription pipelines.

| Category | WER |
|---|---|
| Pure English (Indian vocab) | 4% |
| Pure Hindi | 16% |
| Addresses + PINs | 37% |
| Indian names | 66% |
| Hinglish (romanized) | 100% |

Dataset: [huggingface.co/datasets/Primepluto/hinglish-whisper-benchmark](https://huggingface.co/datasets/Primepluto/hinglish-whisper-benchmark)
